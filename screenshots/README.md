# Screenshots

This directory is for capturing screenshots of the Excel workbook.

## Recommended Screenshots

### 1. Raw Vendor Files
- Show all 5 files in `data/raw/` in Windows Explorer
- Capture the different column names visible in each file
- File: `01_raw_vendor_files.png`

### 2. Master Products Sheet
- Open `data/processed/nexa_parts_master_database.xlsx`
- Navigate to the **Master Products** sheet
- Show the styled header row and first ~20 rows
- Highlight: alternating row fills, conditional formatting on status and low-stock columns
- File: `02_master_products_sheet.png`

### 3. Duplicate Review Sheet
- Navigate to the **Duplicate Review** sheet
- Show a few duplicate groups (SELECTED vs DUPLICATE rows highlighted in green/orange)
- File: `03_duplicate_review_sheet.png`

### 4. Data Issues Sheet
- Navigate to the **Data Issues** sheet
- Show HIGH (red), MEDIUM (yellow), LOW (green) severity rows
- Show the `resolved` dropdown column
- File: `04_data_issues_sheet.png`

### 5. Import Log Sheet
- Navigate to the **Import Log** sheet
- Show all 5 vendor rows with their row counts and issue tallies
- File: `05_import_log_sheet.png`

### 6. Category Summary Sheet
- Navigate to the **Category Summary** sheet
- Show the "Products by Category" and "Top 10 by Stock Value" tables
- File: `06_category_summary_sheet.png`

### 7. Inventory Dashboard Sheet
- Navigate to the **Inventory Dashboard** sheet
- Capture the full KPI cards section (Total Products, Active, Discontinued, etc.)
- Show the summary mini-tables below the KPIs
- File: `07_inventory_dashboard_sheet.png`

---

*Tip: Use Windows Snipping Tool (Win + Shift + S) or ShareX for high-quality captures.*
